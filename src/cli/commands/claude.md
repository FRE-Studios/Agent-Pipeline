# src/cli/commands/

Individual CLI command implementations.

## Files

- **init.ts** - Project scaffolding command (creates example pipelines and agents)
- **rollback.ts** - Git rollback command for reverting pipeline commits
- **analytics.ts** - Pipeline performance metrics and analytics display
- **cleanup.ts** - Pipeline branch cleanup command
- **history.tsx** - Interactive history browser with Ink/React UI
